OBJETIVO PRINCIPAL: 
    - Criação do Dashboards de SAC com uma guia/aba Atendimento, a partir do arquivo em excel "Base SAC".

O PROJETO TERÁ O SEGUINTE TÓPICO:
- Análise de atendimentos aos usuários

VISUALIZAÇÕES E MEDIDAS QUE SERÃO CRIADAS: 
    - Guia ATENDIMENTO - Filtro de ano
    - Guia ATENDIMENTO - Total de Atendimento (Cartão de linha multipla) = Medida Chamados Respondidos = COUNT(fOcorrencias[Data Resposta]) 
    - Guia ATENDIMENTO - Média diária de resposta (Cartão de linha multipla)  = Medida Média Diária de Resposta de Chamados = VAR vDias = DISTINCTCOUNT(fOcorrencias[Data Resposta]) RETURN DIVIDE([Chamados Respondidos], vDias)
    - Guia ATENDIMENTO - Média diária de chamados (Cartão de linha multipla) = Medida Média Diária de Abertura de Chamados = VAR vDias = CALCULATE(DISTINCTCOUNT(fOcorrencias[Data Chamado]), USERELATIONSHIP(fOcorrencias[Data Chamado], Calendario[Data])) RETURN DIVIDE([Chamados Abertos], vDias) 
    - Guia ATENDIMENTO - Média de resposta em dias (Cartão de linha multipla) = Medida Média de Tempo de Resposta em dias = AVERAGEX(fOcorrencias, fOcorrencias[Data Resposta] - fOcorrencias[Data Chamado])
    - Guia ATENDIMENTO - Tempo médio de duração (Cartão de linha multipla) = Medida Média de Duração do Chamado = VAR vTotalMinutos = INT(AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60) VAR vTotalSegundos = INT((AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60  - vTotalMinutos )* 60) RETURN vTotalMinutos & " min e " & vTotalSegundos & " seg"
    - Guia ATENDIMENTO - Tempo total de atendimento (Cartão de linha multipla) = Medidas 
	Total Dias  = 
	VAR vTempoTotal = SUM(fOcorrencias[Tempo Resposta (s)])
	VAR vTotalDias = INT(vTempoTotal)
	VAR vTotalHoras = INT( (vTempoTotal - vTotalDias) * 24)
	VAR vTotalMinutos = INT( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60)
	VAR vTotalSegundos = INT( ( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60 - vTotalMinutos ) * 60 )
	RETURN vTotalDias & " dia(s)"
	Total Horas = 
	VAR vTempoTotal = SUM(fOcorrencias[Tempo Resposta (s)])
	VAR vTotalDias = INT(vTempoTotal)
	VAR vTotalHoras = INT( (vTempoTotal - vTotalDias) * 24)
	VAR vTotalMinutos = INT( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60)
	VAR vTotalSegundos = INT( ( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60 - vTotalMinutos ) * 60 )
	RETURN vTotalHoras & " hora(s)"
	Total Minutos =
	VAR vTempoTotal = SUM(fOcorrencias[Tempo Resposta (s)])
	VAR vTotalDias = INT(vTempoTotal)
	VAR vTotalHoras = INT( (vTempoTotal - vTotalDias) * 24)
	VAR vTotalMinutos = INT( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60)
	VAR vTotalSegundos = INT( ( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60 - vTotalMinutos ) * 60 )
	RETURN vTotalMinutos & " minuto(s)"
	Total Segundos = 
	VAR vTempoTotal = SUM(fOcorrencias[Tempo Resposta (s)])
	VAR vTotalDias = INT(vTempoTotal)
	VAR vTotalHoras = INT( (vTempoTotal - vTotalDias) * 24)
	VAR vTotalMinutos = INT( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60)
	VAR vTotalSegundos = INT( ( ( (vTempoTotal - vTotalDias) * 24 - vTotalHoras ) * 60 - vTotalMinutos ) * 60 )
	RETURN vTotalSegundos & " segundo(s)"
    - Guia ATENDIMENTO - Chamados respondidos e chamados em aberto (Gráfico de Colunas Clustorizados) = Medidas Chamados Abertos = CALCULATE(COUNT(fOcorrencias[Data Chamado]), USERELATIONSHIP(fOcorrencias[Data Chamado], Calendario[Data])), Chamados Respondidos = COUNT(fOcorrencias[Data Resposta])  e Data
    - Guia ATENDIMENTO - Chamados respondidos por problema (Gráfico de Barras Clustorizados) = Medida Chamados Respondidos = COUNT(fOcorrencias[Data Resposta]) e Problemas com tooltip:
	a) Sem título (Cartão) = Medida Primeiro Problema 
	b) Sem título (Gráfico de rosca) = Medida Status x Chamados Respondidos = COUNT(fOcorrencias[Data Resposta]) 
    - Guia ATENDIMENTO - Chamados respondidos por usuário (Gráfico de Barras Clustorizados) = Medida Chamados Respondidos = COUNT(fOcorrencias[Data Resposta]) e Nome do Usuário
    - Guia ATENDIMENTO - % de chamados atendidos (Gráfico de Rosca) = Medidas % de Respostas = DIVIDE([Chamados Respondidos], [Chamados Abertos]) e % em Aberto = DIVIDE([Chamados Abertos] - [Chamados Respondidos],[Chamados Abertos])
    - Guia ATENDIMENTO - Média de tempo de atendimento (min) (grafico de área) = Medida Média Tempo = AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60 e Ano, nome do mês e dia com tooltip:
	a) Tempo médio de resposta (Cartão) = Medida Média de Duração do Chamado = VAR vTotalMinutos = INT(AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60) VAR vTotalSegundos = INT((AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60  - vTotalMinutos )* 60) RETURN vTotalMinutos & " min e " & vTotalSegundos & " seg"
	b) Top 5 problemas do mês x média (Matriz) = Medida Média de Duração do Chamado TOOLTIP = VAR vTotalMinutos = INT(AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60) VAR vTotalSegundos = INT((AVERAGE(fOcorrencias[Tempo Resposta (s)]) * 24 * 60  - vTotalMinutos )* 60) RETURN IF(NOT ISBLANK(vTotalMinutos) && NOT ISBLANK(vTotalSegundos),vTotalMinutos & " min e " & FORMAT(vTotalSegundos, "00") & " seg") e Problema
